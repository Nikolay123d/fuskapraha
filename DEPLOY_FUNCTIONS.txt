MAKÁME CZ — Firebase Functions list (europe-west1)

ВАЖНО:
- НЕ редактируйте firebase/functions/index.js «как список функций» — это код.
- Этот файл просто шпаргалка: какие функции должны быть задеплоены.

CALLABLE (httpsCallable):
- pingServer
- profileInit
- consumeLimit

DM:
- dmRequestSend
- dmRequestDecline
- dmConfirmAtomic
- dmSend

Friends:
- friendSend
- friendAccept
- friendDecline
- friendRemove

Moderation/Admin:
- blockUser
- unblockUser
- reportUser
- supportTicketCreate

Autopost:
- autopostCampaignCreate
- autopostCampaignUpdate
- autopostCampaignToggle
- autopostCampaignRemove

TRIGGERS / SCHEDULER:
- onChatMessageCreate (RTDB trigger)
- promoChat25Tick (Scheduler)
- autopostTick (Scheduler)

DEPLOY COMMANDS:
1) Deploy all functions:
   firebase deploy --only functions

2) Deploy only selected functions (example):
   firebase deploy --only "functions:profileInit,functions:autopostCampaignCreate,functions:autopostCampaignToggle,functions:autopostTick"
